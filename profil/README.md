## Qu'est-ce que c'est ?

Le profil contient vos **regles de collaboration, preferences, architecture de sessions, et specs** — tout ce qui definit comment Claude travaille avec vous.

Fichier cle : `collaboration.md` — vos regles de travail non negociables.

## Setup

Le script `brain-setup.sh` clone automatiquement ce satellite :

```bash
# Automatique via setup
bash scripts/brain-setup.sh

# Ou manuellement
git clone https://github.com/Tetardtek-Cortex/Cortex-Template-Profil.git profil
```

> Forkez [Cortex-Template-Profil](https://github.com/Tetardtek-Cortex/Cortex-Template-Profil) pour avoir votre propre version.
