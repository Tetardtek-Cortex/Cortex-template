## Qu'est-ce que c'est ?

La progression contient vos **metriques de session, metabolism, et historique de sante cognitive** — le suivi de comment vous et votre brain evoluez au fil du temps.

## Setup

Le script `brain-setup.sh` clone automatiquement ce satellite :

```bash
# Automatique via setup
bash scripts/brain-setup.sh

# Ou manuellement
git clone https://github.com/Tetardtek-Cortex/Cortex-Template-Progression.git progression
```

> Forkez [Cortex-Template-Progression](https://github.com/Tetardtek-Cortex/Cortex-Template-Progression) pour avoir votre propre version.
